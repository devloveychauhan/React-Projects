## Getting started (local)
1. Clone the repo
   - git clone https://github.com/devloveychauhan/React-Projects.git
2. Enter the project folder you want to run, for example:
   - cd React-Projects/<project-folder>
3. Install dependencies
   - npm install
   (or `yarn` / `pnpm` depending on the project)
4. Start the dev server
   - npm start
5. Open http://localhost:3000 (or the port shown in the console)

Each project folder contains its own README with project-specific instructions (dependencies, environment vars, API keys).
